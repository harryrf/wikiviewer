Requirements for fCC:

  • Can search Wikipedia entries in a search box and see the resulting Wikipedia entries.

  • Can click a button to see a random Wikipedia entry.


    • Design Ideas:

  • User is presented with two buttons, "search" and "random".

    • Search: User clicks, buttons disappear and a search bar eases in.

      • Search Bar: User enters search, returns, and results are given.


    • Random: User clicks, buttons disappear and a random result is returned.
